# Getting Started with Blogger Auto Publishing

Welcome to the **Blogger Auto Publishing** project! This tool helps you publish blog posts to Blogger automatically using markdown files.

## Why Automate Blogger Publishing?

Managing a blog can be time-consuming. With this tool, you can:

- Write posts in markdown, the simplest formatting language
- Publish to Blogger with a single push to GitHub
- Never worry about Blogger's rich text editor again

## How It Works

1. Write your post in a `.md` file under the `posts/` directory
2. Add metadata like title and labels in the frontmatter section
3. Push your changes to GitHub
4. GitHub Actions automatically publishes your post to Blogger

## Markdown Features

You can use all standard markdown features:

- **Bold** and *italic* text
- [Links](https://www.blogger.com)
- Images
- Code blocks
- Lists and tables
